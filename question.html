Question-3: What is event delegation in the DOM?

Event Delegation is the process of adding event listeners to a parent element so that the event is triggered on all child elements under that parent. It works based on the Event Bubbling mechanism.

Question-4: What is event bubbling in the DOM?

Event Bubbling is when an event occurs on a child element, that event propagates to its parent and further up the hierarchy until it reaches the document.
